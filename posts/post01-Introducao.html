Introdução ao HTML5

HTML em português significa Linguagem de Marcação de Hipertexto. É a linguagem padrão utilizada para criar e estruturar as páginas que vemos na internet.

Vamos entender melhor essas palavras:

  • Linguagem de marcação: Diferente de uma linguagem de programação (como Python, JavaScript), que dá instruções e executa lógicas, uma linguagem de marcação serve para descrever e oranizar o conteúdo. Você usa "etiquetas" (tags) para dizer ao navegador o que cada parte do seu texto representa:"isto é um título", "isto é um parágrafo", etc.
  • Hipertexto: É um texto que contém links (hiperlinks) para outros textos ou documentos. é a funcionalidade que permite que você clique em uma imagem ou palavra e seja levado para outra página ou até mesmo para parte da mesma página, criando a teia deinformações que conhecemos com a World Wide Web.

A evolução do HTML

Atualmente estamos na versão 5 do HTML, por isso ele é chamado de HTML5. Comparado à última versão, houve várias melhorias cruciais:

  • Semântica aprimorada
  • suporte nativo a mídia
  • gráficos e desenhos
  • formulários inteligentes
  • APIs avançadas

Assim, resume-se que o HTML5 é importante porque modernizou a web, tornado-a mais semântica, acessível, interativa e menos dependente de tecnologias de terceiros.

A importância da semântica

Semântica, no contexto do HTML, significa usar as tags de acordo com o seu significado. Usamos <header> para cabeçalhos e <footer> para rodapés, tornando o código mais acessível e otimizado para SEO.
